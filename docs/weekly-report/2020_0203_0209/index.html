まとめ

このブログ作ったので、これからどんどんアウトプットをしていけたらと思う。
作った理由としてはもちろんアウトプットだけど、それ以外にも自分でどれだけやってるかやってきたかを文字にすることで、振り返りがしやすくなるかなというのもある。

今週でようやくクリーンアーキテクチャ本を読了した。
構造化プログラミングやオブジェクト指向プログラミング、関数型プログラミングやさまざまな原則など、前提となる知識が乏しくはじめは挫折したものの、今のプロジェクトでクリーンアーキテクチャのような構成でやっているので、今回の挑戦ではなんとなく理解できたかな。くらいにまではいけた。

TS, React, Reduxの環境でベースとなるものを作った。今更だけど、この構成。
これはしばらくは触らなそうだけど、そのうち多分触ると思う。

エンジニアになって今年が一番モチベーションが上がっている。
今までもちょくちょく自分で作ったり取り組んだりしてたけど、今年は自分でもわかるくらい今までと違う！
もっともっと追求したり、自分で何か作ったりして、このモチベーションが続くようにする。
